# キキョウ発注書 🌸

**登録不要・完全無料・データはどこにも送信されない**発注書ジェネレーターです。
外注・仕入れに使う発注書を、ブラウザだけで作れます。

👉 **今すぐ使う:** https://taku0826.github.io/kikyo-order/

## 特徴

- ✍️ **リアルタイムプレビュー** — 入力した内容がその場で書面に反映されます
- 🖨️ **印刷 / PDF保存** — ブラウザの印刷機能でそのままきれいなPDFに
- 📋 **発注書の必須項目に対応** — 発注番号・発注日・希望納期・納品場所・支払条件・検収条件
- 💴 **税率別の消費税計算** — 10% / 8%(軽減税率)を品目ごとに設定、軽減税率注記も自動表示
- 🔴 **朱印スタンプ** — 発注者の姓から朱印風のハンコを自動生成
- 🔒 **完全ローカル動作** — サーバーに一切データを送信しません。入力内容はあなたのブラウザの中だけ

## 「発注書」とは

発注書は、**あなたが外注・仕入れをする相手(受注者)に宛てて出す**書類です。
受け取る側の書類(請求書・領収書など)とは立場が逆になります。
デザイナーがイラストを外注する、店主が商品を仕入れる、といった場面で使います。

## 使い方

1. ページを開く
2. 発注番号・宛先(受注者)・発注者・品目を入力する
3. 「印刷 / PDF保存」ボタンを押す

それだけです。アカウント登録も、メールアドレスの入力も要りません。

## 仕組み

単一のHTMLファイルだけで動作します。ビルドなし、依存ライブラリなし。
`index.html` をダウンロードしてオフラインで使うこともできます。
